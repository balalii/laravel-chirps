

## Installation

Copy the .env.example

```bash
cp .env.example .env
```

Install dependencies with Composer

```bash
composer install
```

Generate a new key

```bash
php artisan key:generate
```

Run the migrations

```bash
php artisan migrate
```

Install npm

```bash
npm install
```

run node

```bash
npm run dev
```

run server

```bash
php artisan serve
```
